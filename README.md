# PlumAntics Mods — Report a Bug

Something broken in a PlumAntics mod? You're in the right place.

## 🔍 Before You Report (30 seconds!)

Duplicate reports slow us down. **Quick check:**
1. [Search open issues](../../issues) — use your mod name + the problem
2. Check [known issues](docs/known-issues.md) — we list problems we already know about
3. Found it already reported? React with 👍 instead of posting again

**Not finding it?** Go ahead and report below. 👇

---

## Report Your Issue

- **[Bug Report](../../issues/new?template=bug_report.yml)** — Something isn't working right
- **[Compatibility Issue](../../issues/new?template=compatibility_report.yml)** — Two mods fighting with each other
- **[Balance Feedback](../../issues/new?template=balance_feedback.yml)** — Feature feels too strong/weak
- **[Question](../../issues/new?template=question_or_help.yml)** — How do I use this? Is this a bug?
- **[View All Issues](../../issues)** — See what's being tracked

---

## How to Report

1. Pick the issue type above
2. Fill in the form (it's designed for Sims players, not programmers)
3. Hit submit

**What helps us most:**
- What you were doing when it happened
- What went wrong (what did you see?)
- Can you repeat it?
- Screenshot if you have one

---

## Status Labels (Where is your issue?)

| Status | Meaning |
|--------|---------|
| **status:needs-triage** | Just came in, we're looking |
| **status:confirmed** | Real bug, we're working on it |
| **status:needs-info** | We need more details from you |
| **status:fixed-next-patch** | Fix is ready, coming soon |
| **status:released** | Fixed! Update available now |

---

## After You Report

You'll get notifications as the issue moves through the process. We'll comment if we need more info. Just watch for updates — no action needed from you.

---

## FAQ

**Not sure if it's a bug?** Use the [Question form](../../issues/new?template=question_or_help.yml). We'll figure it out.

**Can I upload a screenshot?** Yes! Forms let you drag & drop files. Not required, but helpful.

**My mod isn't in the list?** Pick the closest match or just type it. We have all PlumAntics mods covered.

**What helps most?** Step-by-step: "I clicked X, then Y, then Z happened." That's gold.

---

## Play Nice

This space is for Sims players and creators. Let's keep it friendly, constructive, and spam-free.

- Be respectful to other players and creators
- No drama, venting, or personal attacks
- Stick to PlumAntics mod issues (not general game problems)

If something crosses the line, we'll close it.
